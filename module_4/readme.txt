Проект: Юнит 4, проект 4, Проект 4. Авиарейсы без потерь

Основные цели и задачи проекта:
--  Необходимо подготовить один большой и сложный запрос, который обязательно должен позволить извлечь из базы id рейса, 
название города отправления (можно код аэропорта), название города прилёта (можно только код), информацию о самолётах, 
пассажирах и суммарной стоимости билетов за каждый рейс (по каждому id рейса сумма проданных билетов), длительность рейсов по времени. 
По итогам выгрузить данные в формате *.csv (одна большая таблица) и сохранить запрос в *.txt.
--   Подготовить презентацию с выводами.
--   Анализ данных (с возможным самостоятельным дополненим данных)

Краткая информация о данных: Демонстрационная база данных «Авиаперевозки» https://postgrespro.ru/docs/postgrespro/10/demodb-bookings.

Этапы работы над проектом:
--  Изучение данных - разные запросы для понимания
--  Написание необходимого запроса
--  Создание презентации
--  [к сожалению, в нормальном формате не успел, поэтому отсутствует] самостоятельное дополнение данных и их анализ
